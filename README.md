# Conversation Intelligence Docs Repo

This repository contains the generated documentation for the multi-tenant conversation intelligence system.

## Structure

- `docs/Conversation_Intelligence_PRD.docx` — Product requirements document
- `docs/Conversation_Intelligence_Architecture.docx` — Technical architecture document
- `docs/Conversation_Intelligence_Playbook.docx` — Training, evaluation, and deployment playbook
- `docs/README.txt` — Pack summary from the original generated bundle

## Notes

- The documents are included directly in `docs/` (not zipped).
- This repo was initialized locally in the sandbox with an initial commit for easy download and handoff.
